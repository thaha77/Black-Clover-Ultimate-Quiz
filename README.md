# 🍀 Black Clover Ultimate Quiz 🍀

Welcome to the **Black Clover Ultimate Quiz**, a web-based quiz application created for die-hard anime fans to test their knowledge about the Clover Kingdom! 

Built by **Black Shadow** 😈

---

## 🚀 Features
* **50 Epic Questions:** Covers everything from Asta's swords to the Dark Triad of the Spade Kingdom.
* **Dynamic Counter:** Real-time question tracking (e.g., `Question 1 of 50`) so players know their progress.
* **Neon Dark Theme:** Clean, modern, responsive UI built with a cool neon-cyan styling.
* **Instant Scoring:** Automatically calculates your score and gives you a breakdown at the end with a "Play Again" option.

---

## 🛠️ Tech Stack Used
* **HTML5:** For structuring the quiz layout and buttons.
* **CSS3:** For the dark anime neon aesthetic and smooth hover animations.
* **JavaScript (ES6):** For handling the quiz logic, tracking scores, dynamic question rendering, and array filtering.

---

## 🎮 How to Play
1. Open the project in your browser.
2. Read the question displayed at the top.
3. Click on one of the four options.
4. Try to get a perfect `50/50` score and prove you are worthy of becoming the next Wizard King!
