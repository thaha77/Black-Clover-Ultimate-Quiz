# Black Clover Ultimate Quiz

Hi! This is a simple web quiz game about the anime Black Clover. I built this using HTML, CSS, and basic JavaScript.

## Features
- 50 different questions about Black Clover.
- A live counter that shows "Question 1 of 50".
- Dark anime style layout with neon lights.
- Final score screen with a retry button.

## How to Run
Just open the `index.html` file in any browser and start playing!
